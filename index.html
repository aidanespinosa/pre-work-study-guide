GIT/GIT-HUB

  • cd: change director
  • mkdir: make new directory
  • pwd: print working directory
  • git checkout:leave branch
  • git -b "New branch name": make new branch
  • git status: check current working branch
  • git add: add changes
  • git -A: -A flag means all
  • git commit: commit changes to branch
  • -m"":message flag
  • git pull origin main: sync local branch with remote branch
  • git push origin "name of working branch":push changes on local branch to remote repository
  • git push origin "name of branch": push local branch to remote repository
  • git branch -d "branch name": delete local branch
